# 1.2 Authentication System — UI Setup Instructions
**Phase/Stage:** 1.2 — Authentication System
**Last Updated:** 2026-02-23

## LoginForm Component Layout
ColumnPanel (content_panel)
>OUTLINED Card (card_auth){nested in content_panel}
>>ColumnPanel (col_form) {nested in card_auth}
>>>Heading (lbl_title) {nested in col_form}
>>>Text (lbl_email) {nested in col_form}
>>>TextBox (txt_email) {nested in col_form}[set Writeback property]
>>>Text (lbl_password) {nested in col_form}
>>>TextBox (txt_password) {nested in col_form}[set Writeback property]
>>>Checkbox (chk_remember) {nested in col_form}
>>>FlowPanel (flow_actions) {nested in col_form}
>>>>Button (btn_login) | Spacer | Link (lnk_forgot) {nested in flow_actions}
>>>FlowPanel (flow_links) {nested in col_form}
>>>>Text (lbl_signup_prompt) | Link (lnk_signup) {nested in flow_links}
>>>Text (lbl_error) {nested in col_form}

---

## Component Properties that CANNOT be set programmatically
| Component | Property | Value |
|-----------|----------|-------|
| txt_email | (W) | ✓ |
| txt_password | (W) | ✓ |
| txt_password | Password | ✓ |

---

## Legend:
> = nesting level
| = Separates components on the same horizontal
OUTLINED = stated when a component is outlined eg: OUTLINED Card; OUTLINED Button
